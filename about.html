Pentru blatul de cocos:

  • Apasa-ma
  • 150 gr zahar
  • putina sare
  • 100 gr nuca de cocos
  • 125 gr faina
  • esenta vanilie

Pentru blatul de cacao:

  • 4 oua
  • 4 linguri ulei
  • 100 gr zahar
  • 100gr faina
  • 50 gr cacao
  • putina sare
  • esenta rom
  • o lingurita praf de copt

Pentru Crema

  • 6 galbenusuri
  • 150 gr zahar
  • 200 gr unt
  • 6 linguri lapte
  • esenta vanilie

Pentru glazura

  • 200 gr smantana pentru frisca
  • 200 gr ciocolata fondente
  • 50 gr unt
    •

    Mod de preparare

    Se separa albusurile de galbenusuri.

    Albusurile se mixeaza cu putina sare pana se fac spuma, se adauga zaharul mixandu-se in continuare, pana se topeste zaharul moment in care se adauga faina, nuca de cocos si vanilia si se amesteca cu o paleta de jos in sus.

    O tava cu dimensiunea de 30/22 cm se tapeteaza cu hartie de copt si se pune compozitia blatului de cocos nivelandul cu o paleta.

    Se da tava la frigider si se face rapid si blatul de cacao.

    Preparam blatul de caco

    Pentru blatul de cacao se separa albusurile de galbenusuri.

    Albusurile se mixeaza cu putina sare pana se fac spuma se adauga, apoi zaharul si se mixeaza pana se topeste zaharul, apoi se adauga galbenusurile, uleiul si esenta de rom dupa care se pune faina amestecata cu cacao si praful de copt si se amesteca cu o paleta pana se incorporeaza toate.

    Se scoate tava cu blatul de cocos din frigider se pune deasupra compozitia blatului cu cacao si se niveleaza cu o paleta. Se coace la 180 de grade Celsius pentru 25-30 minute. Se face testul cu scobitoarea pentru a vedea daca este copt.Apoi se lasa la racit.

    Preparam crema prajiturii

    Galbenusurile se mixeaza cu zaharul si laptele si se pun la fiert la bain marie pana se ingroasa la final se pune vanilia se amesteca si se lasa la racit.

    Untul la temperatura camerei se mixeaza pana devine spumos apoi se adauga cate o lingura de crema de galbenusuri si se mixeaza in continuare.

    Se procedeaza asa pana se pune toata crema de galbenusuri.

    Crema obtinuta se pune peste blat se niveleaza si se pune la frigider pana se face glazura.

    Pregatim si glazura

    Smantana pentru frisca se incalzeste se adauga ciocolata maruntita si untul se fierb pentru 5 minute la foc mic apoi se lasa la racit.

    Dupa ce glazura s-a racit se pune peste crema se niveleaza si se pune la frigider pentru cateva ore.

    Atentie nu puneti glazura calda peste crema!

    Ideal ar fi de seara pana dimineata. Dupa cateva ore se taie si se orneaza cu nuca de cocos.

    Asta este totul, prajitura noastra este gata, va invit cu drag si pe voi sa incercati reteta si de abia astept sa reveniti, sa ne spuneti si noua cum a iesit.

    Pana atunci, nu imi ramane decat sa va urez mult spor la gatit si Pofta buna!
